Description:
Herbal Medicine Checker is a full-stack web application built with React (frontend) and Python Flask (backend). It allows users to quickly check herbal products for their active ingredients and potential warnings. This project demonstrates state management in React, API communication, and backend data handling — making it a practical portfolio project for developers in the healthcare and technology space.

Features:

User-friendly React form for entering herbal product names

Backend API built with Flask that processes requests and returns product information

Dynamic display of active ingredients and safety warnings

Handles unknown products gracefully with error messages

Fully cross-origin compatible with CORS enabled

Easy to expand with databases or additional products

Tech Stack:

Frontend: React, JavaScript, HTML, CSS

Backend: Python, Flask, Flask-CORS

Data: Python dictionary (can be replaced with database for scalability)

Usage:

Clone the repository

Run Flask backend: py backend/app.py

Run React frontend: npm start inside my-app folder

Enter a herbal product name and click Check to see its ingredients and warnings

Demonstrates real-world full-stack development

Practical application relevant to pharmacy and herbal medicine
